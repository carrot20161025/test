# 安装脚手架-初始化项目

angular官网提供脚手架，需要全局安装

1. 安装脚手架

   ```
   npm i -g @angular/cli
   ```

2. 在某个**自定义的目录**下**初始化一个项目**

   ```
   ng new my-app
   ```

3. 进入这个新项目

   ```
   cd my-app
   ```

4. 启动angular项目

   ```
   ng serve    //默认打开localhost:4200  ，不会自动打开浏览器
   ng serve -o  //会自动打开浏览器
   ng serve -open  //与上面一致
   ```
